To play Jump Death:
  1. Install "XNA DirectX Framework 4.0"
  2. Install "XNA Framework 4.0 Redistribution"
  3. Run "setup.exe"

To run Jump Death in Visual Studio:
  1. Install "XNA DirectX Framework 4.0"
  2. Install "XNA Framework 4.0 Redistribution"
  3. Install "XNA Game Studio 4.0 Platform Tools"
  4. Install XNA Game Studio 4.0 Shared"
  *  For Visual Studio > 2010
     5. Install appropriate Visual Studio Extension
